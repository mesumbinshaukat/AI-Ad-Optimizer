Deploying this project to Vercel is straightforward since we have included a standard `package.json` and `vite.config.ts`.

### 1. Push to GitHub
Upload all files in this directory to a new GitHub repository.

### 2. Connect to Vercel
1. Go to [vercel.com](https://vercel.com) and click **"Add New"** > **"Project"**.
2. Select your GitHub repository.
3. Vercel will auto-detect the **Vite** framework.

### 3. Setup Environment Variables
Before clicking "Deploy", you must add your API Key:
1. In the **Environment Variables** section:
   - **Key**: `API_KEY`
   - **Value**: `[Your Gemini API Key]`
2. Click **"Add"**.

### 4. Deploy
Click **"Deploy"**. Vercel will build the project and provide a live URL (e.g., `ad-genius-ai.vercel.app`).
